What are we good at?

UI / UX Design

UX design - our role is to make your product user-friendly and accessible. We get to know your target audience, identify the problems that need to be solved and come up with solutions to those problems.

UI design - our job is to design all the screens that the user will navigate through, and to create visual elements and their interactive properties that facilitate this movement.

HTML, CSS, JavaScript

With HTML we create the structure of the website and with CSS we create the design and adaptive layout of the website

We use JavaScript in web development to add interactivity and features to improve the user experience and make websites much more user-friendly.

Sass, GreenSock, React

Why do we use Sass? It allows us to use features that CSS doesn't, such as variables, nested rules, mixins, imports, inheritance, built-in functions and more.

Greensock allows us to animate just about anything that can be accessed with JavaScript, shared objects, canvases and more.

With React, we design and build JavaScript-based applications for web or mobile environments.
